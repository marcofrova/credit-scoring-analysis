# Credit Card Application Scoring

### Overview
This project implements a complete credit risk pipeline, featuring a **Logistic Regression from scratch** using IRLS and a **Bayesian approach** with Metropolis-Hastings.

- **Custom IRLS Solver**: Implementation of MLE for Logistic Regression.

- **Bayesian Inference**: Posterior analysis via MCMC for robust coefficient estimation.
- **Three-Way Policy**: Triage system (Auto-approve / Manual Review / Decline).
